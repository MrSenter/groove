## 分流规则

先读 `快速入口.md`，判断是否命中已知工作流并查看其流畅等级：

| 等级 | 本次读取 |
|---|---|
| A 完全稳定 | 执行卡片 + 必要公共文档 |
| B 基本稳定 | +候选规则/样例(按需) + 巡查日志最新一份相关段落 |
| C 已做过未稳定 | workflow.md + candidates(按需) + 巡查日志最新一份相关段落 |
| D 未命中 | `Agent执行手册.md` 新任务流程 |

没命中或要改规则 → `Agent执行手册.md`（新任务/异常）或 `完整工作手册.md`（规则/结构）。

首次进入本工作区先看一次 `系统索引/工作区地图.md`。

记录位置、越用越省 token 闭环、A/B/C/D 详解都在 `完整工作手册.md`，不在本文件展开。
